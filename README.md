## Ejecución local

Ir al directorio del proyecto

```bash
  cd TP1-PROTOS
```

Ejecutar make

```bash
  make all
```

Iniciar servidor

```bash
  ./server (puerto)
```

Enviar datagramas por UDP

```bash
  nc -u (ip) (puerto)
```

Conectarse por TCP

```bash
  nc -c (ip) (puerto)
```

Para remover la instalación

```bash
  make clean
```
